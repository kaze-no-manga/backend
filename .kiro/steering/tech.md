# Technology Stack - Backend

## Core
- AWS AppSync (GraphQL)
- AWS Lambda (Node.js 20.x)
- TypeScript 5.x

## AWS Services
- AppSync - GraphQL API
- Lambda - Compute
- Step Functions - Workflows
- SQS - Queue
- SES - Email
- SNS - Push notifications
- DynamoDB - Cache
- RDS - Database (via database repo)

## Build
- esbuild (Lambda bundling)
- AWS CDK (deployment)
